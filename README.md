## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ (for local server)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Muri-tuu/M-Solutions-Website.git
cd M-Solutions-Website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run serve
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
